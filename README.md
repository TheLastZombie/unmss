# unmss

unmss is a command-line open source converter for Advanced Mario Sequencer (.MSS) files.

## Requirements

- [Node.js](https://nodejs.org/)
- [Fluidsynth](http://www.fluidsynth.org/)

## Usage

```
npm start input.mss output.wav
```
